# Finance_Analysis-Power_BI-
The main objective of this project is to provide a single view of financial data so stakeholders can quickly see which categories, regions, and customer segments drive the most revenue and cost.
The report also highlights currency distribution and order status, helping finance teams monitor global performance and transaction health.

Dataset Description
The dataset represents transactional finance data with fields related to products, geography, customers, and currencies.
Typical columns include: Category, Amount, Balance, UnitPrice, Cost, Segment (Enterprise/Retail), Region, Country, Currency (INR, USD, GBP, EUR, etc.), Status (Completed/Failed/Pending), and City.

Dashboard Features
The Finance Analysis dashboard contains the following key visuals and KPIs:
KPI cards showing overall Sum of Balance, Sum of UnitPrice, Sum of Amount, and Sum of Cost.

Bar or column charts for:

Sum of Amount by Category.

Sum of Balance by Category.

Charts for Count of Region by Currency and Count of Cost by Country.
A map visual showing Country and Status (Completed, Failed, Pending) for global monitoring.
Slicers for Segment, Category, Currency, and Status to enable interactive exploration.

Key Insights
Using this dashboard, users can answer questions such as:

Which product categories generate the highest total amount and balance.
How financial metrics differ between Enterprise and Retail segments.
How costs and transactions are distributed across regions, countries, and currencies.
What proportion of orders are Completed, Pending, or Failed in each geography.
